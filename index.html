1. Navigating the Filesystem

Understanding how to move around your Linux system is fundamental.

  • pwd: pwd

    Prints the name of the current/working directory.

  • cd: cd [directory]

    Changes the current directory. Use cd .. to go up one level, and cd ~ or just cd to go to your home directory.

  • ls: ls [options] [directory]

    Lists directory contents. Useful options include ls -l (long listing), ls -a (all files, including hidden ones), and ls -h (human-readable file sizes).

2. Working with Files and Directories

Creating, viewing, and manipulating files and directories are essential tasks.

  • mkdir: mkdir [directory_name]

    Creates a new directory.

  • touch: touch [file_name]

    Creates an empty file or updates the timestamp of an existing file.

  • cat: cat [file_name]

    Displays the content of a file.

  • cp: cp [source] [destination]

    Copies files or directories. Use cp -r for recursive copying of directories.

  • mv: mv [source] [destination]

    Moves or renames files or directories.

  • rm: rm [file_name]

    Removes files. Use with caution! Use rm -r to remove directories and their contents recursively (also use with caution!).

3. Getting Help

Linux has built-in documentation to assist you.

  • man: man [command_name]

    Displays the manual page for a given command. Use the arrow keys to navigate and press q to quit.

  • --help: [command_name] --help

    Many commands offer a brief help message with common options.

This is just a brief introduction to some fundamental Linux commands. There's much more to explore!
